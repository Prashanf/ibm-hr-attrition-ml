---

## 📌 Problem Statement

Employee attrition is a critical HR challenge. Replacing an employee costs **50–200% of their annual salary** when accounting for recruitment, onboarding, and lost productivity. This project builds a machine learning system that identifies employees at risk of leaving — enabling HR teams to intervene proactively before talent is lost.

**Business Question:** *Given an employee's profile, can we predict whether they will leave the company?*

## 🔬 Project Pipeline

### 1. 🔍 Data Inspection
- Shape, dtypes, statistical summary
- Class imbalance analysis (84/16 split)
- Constant column detection and removal (`EmployeeCount`, `Over18`, `StandardHours`)
- Zero missing values confirmed

### 2. 🧹 Data Cleaning
- Dropped 4 uninformative columns (zero variance / ID columns)
- Verified no duplicates or null values

### 3. ⚙️ Feature Engineering
| Engineered Feature | Description |
|---|---|
| `SatisfactionScore` | Composite of Job + Environment + Relationship satisfaction |
| `TenureRatio` | YearsAtCompany / (TotalWorkingYears + 1) — loyalty signal |
| `PromotionLag` | YearsSincePromotion − YearsInCurrentRole — frustration proxy |
| `IsJunior` | Flag: JobLevel ≤ 2 |
| `LowSatisfaction` | Flag: SatisfactionScore < 2.5 |
| `IncomePerYear` | MonthlyIncome × 12 |

**Encoding strategy:**
- `BusinessTravel` → Ordinal (0 / 1 / 2)
- `OverTime`, `Gender` → Binary
- `Department`, `JobRole`, `MaritalStatus`, `EducationField` → One-hot

### 4. 📈 Exploratory Data Analysis
- Feature-target correlation analysis
- Attrition breakdown by Department, JobRole, MaritalStatus, JobLevel
- OverTime, Distance from Home, and Previous Companies risk analysis
- Monthly Income and Tenure distribution by attrition

### 5. 🎯 Feature Selection
- Random Forest Gini importance scores across all 53 engineered features
- Top **20 features** selected for model training
- Visualised importance ranking with selection cutoff

### 6. 🤖 Model Training — 3 Algorithms

| Model | Notes |
|---|---|
| **Logistic Regression** | L2 regularisation (C=0.1), scaled features, `class_weight='balanced'` |
| **Random Forest** | 150 estimators, max_depth=10, `class_weight='balanced'` |
| **Gradient Boosting** | 150 estimators, LR=0.05, subsample=0.8, min_samples_leaf=10 |

All models use `class_weight='balanced'` to address the minority class imbalance.

### 7. ⚡ Hyperparameter Tuning
- `GridSearchCV` with `StratifiedKFold` (3 folds) on Gradient Boosting
- Optimised for **ROC-AUC**
- Parameters tuned: `n_estimators`, `learning_rate`, `max_depth`, `min_samples_leaf`, `subsample`
- 32 parameter combinations × 3 folds = **96 total fits**

## 📉 Results

### Model Comparison

| Model | ROC-AUC | F1 Score | Recall | Precision | Avg Precision |
|---|---|---|---|---|---|
| Logistic Regression | 0.7962 | 0.4472 | **0.7660** | 0.3158 | 0.5864 |
| Random Forest | 0.7846 | 0.3614 | 0.3191 | 0.4167 | 0.3976 |
| Gradient Boosting | 0.7724 | 0.2687 | 0.1915 | 0.4500 | 0.4602 |
| **GB (Tuned)** | 0.7637 | 0.3385 | 0.2340 | — | 0.4777 |

> **Key insight:** Logistic Regression achieved the highest Recall (0.766), meaning it correctly identified 76.6% of employees who actually left — the most important metric in an HR context where the cost of missing a leaver is high.

### Top Attrition Risk Factors

1. 🕐 **OverTime** — strongest single predictor
2. 💰 **Monthly Income** — lower income = higher risk
3. 🎂 **Age** — younger employees are more mobile
4. 🏢 **Total Working Years** — less experienced = higher risk
5. 📍 **Distance from Home** — long commutes correlate with attrition
6. 😐 **Satisfaction Score** — composite of 3 satisfaction dimensions
7. 👔 **Job Role** — Sales Reps & Lab Technicians at highest risk (>20%)
8. 💍 **Marital Status** — single employees leave more

## 💡 Business Recommendations

Based on the analysis:

1. **Audit overtime policies** — especially in Sales and R&D departments
2. **Salary benchmarking** — focus on JobLevel 1-2 (junior roles)
3. **Improve onboarding** — highest attrition in first 1-3 years at company
4. **Career path clarity** — Sales Representatives have >25% attrition rate
5. **Flexible work options** — consider remote/hybrid for employees commuting 20+ km
6. **Satisfaction monitoring** — implement quarterly pulse surveys targeting composite score
